Introduction

ผมทำโปรเจกต์นี้ขึ้นมาเพราะอยากฝึกการเขียน HTML, CSS และ JavaScript ด้วยตัวเองครับ

รู้สึกว่าข้อมูลพื้นฐานที่หาได้ตามเว็บค่อนข้างน้อยและเข้าใจยากสำหรับคนเริ่มต้น จึงอยากรวมเนื้อหาและตัวอย่างโค้ดไว้ให้เข้าใจง่าย ๆ

หวังว่าจะช่วยให้คนที่เริ่มเรียนการเขียนเว็บเข้าใจได้เร็วขึ้น และเป็นการทบทวนความรู้ของผมเองด้วยครับ

ส่วนประกอบหลัก

<!DOCTYPE html> : ประกาศว่าเป็น HTML5

<html lang="th"> : บอกว่าเนื้อหาเป็นภาษาไทย

<head> : ใส่ข้อมูลเกี่ยวกับหน้า (meta, title, link)

<body> : ส่วนที่แสดงผลให้ผู้ใช้เห็น

example-code 

<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>ตัวอย่าง</title>
  </head>
  <body>
    <h1>สวัสดี</h1>
  </body>
</html>

แท็กเชิงความหมาย

<header> : ส่วนหัวของเว็บ เช่น โลโก้ และเมนู

<nav> : แถบเมนูนำทาง

<main> : เนื้อหาหลักของหน้า

<section> : แบ่งเนื้อหาเป็นส่วนๆ

<article> : บทความหรือเนื้อหาที่แยกได้

<aside> : เนื้อหาข้าง ๆ เช่น แถบด้านข้าง

<footer> : ส่วนท้าย เช่น ที่อยู่ ลิขสิทธิ์

example-code 

<header>...</header>
<nav>...</nav>
<main>...</main>
<section>...</section>
<article>...</article>
<aside>...</aside>
<footer>...</footer>

รายการ

<ul> : รายการมีจุด (unordered list)

<ol> : รายการมีตัวเลข (ordered list)

<li> : รายการแต่ละข้อ

example-code 

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

ข้อความ

<h1>-<h6> : หัวข้อใหญ่→เล็ก

<p> : ย่อหน้า

<br> : ขึ้นบรรทัดใหม่ (self-closing)

<strong> : ตัวหนา (เน้นความสำคัญ)

<em> : ตัวเอียง (เน้น)

<span> : ครอบข้อความสั้น ๆ เพื่อใส่สไตล์

<div> : กล่องรวมเนื้อหา

example-code 

<h1>หัวข้อหลัก</h1>
<p>ย่อหน้านี้เป็นตัวอย่าง</p>
<br>
<strong>เน้นข้อความ</strong>
<em>เน้นด้วยตัวเอียง</em>
<span>ข้อความสั้นๆ</span>
<div>ส่วนกล่องแบ่งเนื้อหา</div>
