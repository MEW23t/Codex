Introduction

ผมทำโปรเจกต์นี้ขึ้นมาเพราะอยากฝึกการเขียน HTML, CSS และ JavaScript ด้วยตัวเองครับ

รู้สึกว่าข้อมูลพื้นฐานที่หาได้ตามเว็บค่อนข้างน้อยและเข้าใจยากสำหรับคนเริ่มต้น จึงอยากรวมเนื้อหาและตัวอย่างโค้ดไว้ให้เข้าใจง่าย ๆ

หวังว่าจะช่วยให้คนที่เริ่มเรียนการเขียนเว็บเข้าใจได้เร็วขึ้น และเป็นการทบทวนความรู้ของผมเองด้วยครับ
